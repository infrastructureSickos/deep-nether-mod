# Deep Nether

A custom Minecraft 1.20.1 Forge mod for the InfrastructureSickos server.

## Overview

Deep Nether expands the Nether dimension with new biomes, blocks, mobs, and mechanics.

## Requirements

- Minecraft 1.20.1
- Forge 47.x

## Building

```bash
./gradlew build
```

The compiled jar will be in `build/libs/`.

## Development Setup

1. Clone this repository
2. Run `./gradlew genEclipseRuns` or `./gradlew genIntellijRuns`
3. Open in your IDE and run the `runClient` configuration
